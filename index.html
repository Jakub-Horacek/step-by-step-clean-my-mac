Step by Step Guide to Clean Your MacOS System Data

Clean Mac Manually

  1. Clear system cache

    Go to: Finder > Go > Go to Folder (1), then type in ~/Library/Caches and hit enter. Select all the files and folders inside the Caches folder and delete them.

  2. Clear system logs

    Go to: Finder > Go > Go to Folder (1), then type in /var/log and hit enter. Select all the files inside the Log folder and delete them.

  3. Remove unused language files

    Go to: Finder > Go > Go to Folder (1), then type in /Library/Languages and hit enter. Delete all the language folders you don't need.

  4. Uninstall unused apps

    Go to the: Applications(2) folder and delete the apps you don't use.

  5. Clean up system files

    Use a system cleaning tool like CleanMyMac to scan and remove unnecessary system files.

Clean Mac Using Some 3rd Party Tools

You can use some tool like DaisyDisk to do these steps mentioned above more comfortably with a graphical interface. Alternatively, you can use the TrashTheCache GitHub app by mylesotoole.

Bonuses

dscl . -read /Users/`id -un` JPEGPhoto | tail -1 | xxd -r -p > ~/Desktop/pic.jpg Copy/paste this command into Terminal, hit return, and a file with your user profile picture will be created on your Desktop named pic.jpg

Notes

(1) You can also use the shortcut ⌘Cmd + ⬆Shift + G to open the "Go to Folder" dialog. (while in Finder)

(2) You can also use the shortcut ⌘Cmd + ⬆Shift + A to open the Applications folder. (while in Finder)
