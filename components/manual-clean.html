Clean Mac Manually

  1. Clear system cache

    Go to: Finder > Go > Go to Folder (1) then type in ~/Library/Caches and hit enter. Select all the files and folders inside the Caches folder and delete them.

  2. Clear system logs

    Go to: Finder > Go > Go to Folder (1) then type in /var/log and hit enter. Select all the files inside the Log folder and delete them.

  3. Remove unused language files

    Go to: Finder > Go > Go to Folder (1) then type in /Library/Languages and hit enter. Delete all the language folders you don't need.

  4. Uninstall unused apps

    Go to the: Applications (2) folder and delete the apps you don't use.

  5. Clean up system files

    Use a system cleaning tool like {cleanMyMacLink} to scan and remove unnecessary system files.
